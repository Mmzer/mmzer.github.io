css的margin值auto在水平格式化下的探究

发表于 2016-06-14   |   分类于 前端开发 ， 经验总结   |  

水平格式化

水平格式化有七个属性组成:

  • margin-left
  • padding-left
  • border-left
  • width
  • border-right
  • padding-right
  • margin-right

而且七个属性的值的总和要等于父元素的宽度。

其中，可以设置为auto值的属性有三个：width,margin-left和margin-right；

如果这三个值中的某个值设置为auto，而余下的两个属性设置为特定的值，那么设置为auto的属性会自动确定所需的长度，从而使元素框的宽度等于父元素。

小结

  1. 水平格式化的七个属性值的总和必须等于包含快的宽度。

  2. 可以设置为auto值的属性有三个：width,margin-left和margin-right；

  3. 三个值中的某个值设置为auto，而余下的两个属性设置为特定的值，那么设置为auto的属性会自动确定所需的长度，从而使元素框的宽度等于父元素

  4. 如果左右边距同时设置为auto，且此时元素的宽度固定，则会表现出水平居中的布局

  5. 如果设置了width和左右外边距之一为auto，那么设置了auto值的外边距被重置为0

  6. 三个值都为auto时，width自动填充包含快，左右外边距重置为0

  7. 水平外边距不会出现合并现象。
