Object.assign()

Object.assign()方法可以把任意多个的源对象字数的可美居属性拷贝给目标对象，然后返回目标对象。

语法：

参数：

target:目标对象

sources：任意多个源对象

描述：

Object.assign()方法只会拷贝源对象自身的并且可枚举的属性到目标对象上。该方法使用源对象的 [ [ Get ] ] 和目标对象的 [ [ Set ] ]，所以它会调用相关 getter 和 setter。它分配属性不仅仅是复制或定义新的属性，如果合并源包含了getter,那么该方法就不适合将新属性合并到原型里。

注意：

1. 如果目标对象的某个猪肚属性和源对象的某个属性同名，这时该方法会抛出异常。拷贝过程会被中断，已经拷贝成功的属性不受影响，还没拷贝出来的属性则不会再被拷贝。

2. Object.assign方法会跳过那些值为null或undefined的源对象。

3. Object.assign方法并不是深拷贝的方式

4、继承属性和不可枚举属性不能被拷贝

5、拷贝之后，目标对象自身也会改变。

Object.defineProperties()

在一个对象上添加或修改一个或者多个自有属性，并返回该对象。

Object.defineProperty()

直接在一个对象上定义一个新属性，或者修改一个已经存在的属性， 并返回这个对象。

Object.entries()

Object.entries方法返回一个包含由给定对象所有可枚举属性的属性名和属性值组成的 [属性名，属性值] 键值对的数组，数组中键值对的排列顺序和使用for…in循环遍历该对象时返回的顺序一致（两者的主要区别是 for-in 还会遍历出一个对象从其原型链上继承到的可枚举属性）。

Object.freeze()

冻结一个对象。

冻结指的是不能向这个对象添加新的属性，不能修改其已有属性的值，不能删除已有属性，以及不能修改该对象已有属性的可枚举性、可配置性、可写性。该方法返回被冻结的对象。
